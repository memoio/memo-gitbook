# MEMO 文档

欢迎来到 MEMO 官方文档库。

MEMO 由 [MemoLabs](https://memolabs.org/) 维护。MEMO 是一个人工智能驱动的去中心化平台，正在彻底改变数据存储和管理。通过提供安全、智能和以用户为中心的云解决方案，MEMO 使用户能够以前所未有的方式控制他们的数据。

这些文档是关于 MEMO 系统架构和生态模块的。
